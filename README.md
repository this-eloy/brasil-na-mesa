<p>Projeto desenvolvido para fins acadêmicos na disciplina de <b>Desenvolvimento de Páginas Web Dinâmicas</b> do curso de Análise e Desenvolvimento de Sistemas no <b>IFSP</b>.</p>

<p>O Brasil Na Mesa é um website que celebra a gastronomia brasileira, utilizando tecnologias front-end para proporcionar uma experiência interativa e informativa sobre receitas e culinária regional.</p>

<h2>Tecnologias Utilizadas</h2>
<ul>
  <li><b>HTML5:</b> Estrutura semântica e acessibilidade.</li>
  <li><b>CSS3:</b> Design responsivo, animações e layouts com Flexbox/Grid.</li>
  <li><b>JavaScript:</b> Manipulação dinâmica do DOM, uso de localStorage para persistência de dados, interatividade e lógica de negócio.</li>
</ul>

<h2>Funcionalidades</h2>
<ul>
  <li><b>CRUD de Receitas:</b> Permite a adição, edição e exclusão de receitas (funcionalidade para utilizadores logados).</li>
  <li><b>Favoritos:</b> Sistema para favoritar receitas utilizando localStorage.</li>
  <li><b>Interatividade:</b> Sessão de comentários e avaliação de pratos.</li>
</ul>
